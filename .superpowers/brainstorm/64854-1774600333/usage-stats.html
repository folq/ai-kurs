Price Data in model-selectors.ts

// Prices per 1M tokens (USD)
export const MODEL_PRICING = {
  "openai/gpt-4o-mini":     { input: 0.15,  output: 0.60 },
  "openai/gpt-4o":         { input: 2.50,  output: 10.00 },
  "openai/gpt-5.4":        { input: 10.00, output: 30.00 },
  "anthropic/claude-haiku": { input: 0.25,  output: 1.25 },
  // ... etc for all models
} as const;

// Helper
export function calculateCost(modelId, promptTokens, completionTokens) {
  const p = MODEL_PRICING[modelId];
  if (!p) return null;
  return (promptTokens * p.input + completionTokens * p.output) / 1_000_000;
}

How tokens/sec is measured

Streaming pages (prompting, agent): Track the time from first token to last token. Divide output token count by elapsed seconds.

Non-streaming (structured outputs): Show total wall-clock time instead of tok/s, since the response arrives all at once.
